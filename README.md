# 📝 React Todo List App

A simple and clean Todo List app built with **React** that allows you to:

- Add new tasks
- Edit existing tasks
- Update edited tasks
- Delete individual tasks
- Clear all tasks at once

## 🚀 Features

- ✅ Add Todo
- ✏️ Edit Todo
- 🔄 Update Todo
- ❌ Delete Todo
- 🧹 Delete All Todos
